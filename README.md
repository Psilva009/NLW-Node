# NLW - NODE 
Projeto desenvolvido na NLW de 2021

**como executar o projeto**

1. tenha instalado na máquina o [yarn](https://classic.yarnpkg.com/en/docs/install/)e [node.js](https://nodejs.org/en/download/)

2. execute `yarn install` para instalar as dependências

3. execute `yarn start` para iniciar o servidor
